# Network Traffic Analyzer

## Overview
Network Traffic Analyzer is a cybersecurity project developed using Python to capture and analyze network traffic in real time and from PCAP files.

The system extracts packet information, monitors active connections, detects suspicious behavior, and displays results through a modern SOC-style dashboard.

---

## Features
- Live network traffic analysis
- PCAP file analysis
- Protocol detection (TCP, UDP, ICMP)
- Packet and byte statistics
- Top Talker detection
- Active connection monitoring
- Suspicious activity detection
- Real-time dashboard visualization

---

## Technologies Used
- Python
- Scapy
- Flask
- HTML
- CSS

---

## Detection Features
- Possible DoS Detection
